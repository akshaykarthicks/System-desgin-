# System Design Learning Hub

Welcome! This repository is a personal collection of notes, lessons, and practical examples designed to help learn and master core system design concepts. Whether you're preparing for an interview or just looking to build more-scalable systems, you'll find helpful resources here.

## Topics Covered

This repository is organized by topic. Each module contains detailed notes, diagrams, code examples, and often a quiz to test your understanding.

### Core Concepts

* **Load Balancing**
* Caching
* Proxies (Forward vs. Reverse)
* Rate Limiting
* CAP Theorem
* Consistent Hashing

### Database & Data Storage

* Database Sharding / Partitioning
* Database Replication
* SQL vs. NoSQL
* Indexes

### Communication & Integration

* Message Queues
* APIs (REST, gRPC, GraphQL)
* Architectural Patterns (Monolith vs. Microservices)

### Scalability & Reliability

* Fault Tolerance & High Availability
* Content Delivery Network (CDN)
* Domain Name System (DNS)
